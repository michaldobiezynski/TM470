Using WoodMac Parking

This is a parking application for Wood Mackenzie Edinburgh office.

If you would like to gain access, please email Facilities Manager of Edinburgh office.



The following are the rules for users of this application:

1. You have to have an account to book a space.

2. You can only book one space per week. *

3. You can only book a space one day in advance. *

4. If spaces on the map have green background it means they are free.

5. If spaces on the map have red background it means they are booked.

6. You can book a space via the map by clicking on a space with green background.


* Under certain circumstances you can acquire a fixed space which you can use everyday without a need to book it. To find out more, please email Facilities Manager of Edinburgh office.
